# 自然化写作

## 原则

1. 先确定读者、用途和语气；用户提供写作样本时，以样本为最高优先级。
2. 保留事实、数字、引用、专有名词和立场，不为了“有人味”虚构个人经历。
3. 删除套话，再调整节奏；自然不等于故意写乱。
4. 技术文档优先准确，营销文案优先具体，私人表达优先贴近用户原声。

## 重点检查

- 夸大意义：里程碑、关键转折、时代浪潮、重要见证。
- 宣传腔：令人惊叹、丰富多彩、无缝、强大、必看。
- 模糊权威：专家认为、业内人士指出、研究表明但没有来源。
- 模板结构：挑战与展望、泛泛的积极结尾、重复总结。
- AI 高频词和过度连接词：此外、至关重要、深入探讨、彰显、赋能。
- 机械句式：连续三项排比、反复“不是 X 而是 Y”、假范围“从 X 到 Y”。
- 形式痕迹：每句一样长、标题过密、粗体过多、emoji 堆叠、破折号泛滥。
- 对话残留：好问题、让我们开始、希望对你有帮助、需要我继续吗。
- 过度迎合与空洞谨慎：你完全正确、可能在某种程度上也许。

## 流程

1. 阅读原文和可选语气样本。
2. 标记事实不可改动项与最明显的模板痕迹。
3. 用具体名词和直接动词重写，合并重复段落。
4. 混合长短句，但保持逻辑自然。
5. 朗读式检查：删掉不会在真实交流中说出口的句子。
6. 核对数字、引用与结论没有变化。

默认只交付最终稿和一段很短的修改说明。用户要求审计时，再列出原文中的 AI 痕迹。编辑文件时先读取，再用 `write_file` 更新；不要静默改变含义。
